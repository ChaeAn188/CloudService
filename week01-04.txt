📚 클라우드 서비스 & Git 학습 노트

## 1. 기초 리눅스 명령어 (터미널 조작)
* **ls**: 폴더 목록 확인 (\`-a\` 숨김 파일, \`-l\` 상세 정보)
* **cd**: 폴더 이동 (\`..\` 상위로, \`폴더명\` 하위로)
* **pwd**: 현재 경로 확인
* **mkdir**: 새 폴더 만들기
* **touch**: 빈 파일 생성 또는 파일 시간(atime, mtime) 갱신
  - *Tip*: 시스템 정리 대상에서 제외시키거나 백업 트리거용으로 사용
* **cat**: 파일 내용 확인
* **clear**: 화면 지우기
* **history**: 입력했던 명령어 목록 보기

## 2. Git 로컬 관리 (내 컴퓨터)
* **git config**: 사용자 이메일/이름 등록 (최초 1회)
* **git init**: 저장소 시작 (.git 생성)
* **git branch -m main**: 기본 브랜치명을 master에서 main으로 변경
* **git status**: 현재 변경 상태 감지
* **git add .**: 변경 사항을 스테이지(장바구니)에 담기
* **git commit -m "메시지"**: 해시코드와 함께 기록 남기
* **git log**: 커밋 히스토리 확인
* **git checkout [해시코드]**: 과거 시점으로 시간여행

## 3. GitHub 원격 연동 (온라인)
* **git remote add origin [URL]**: 원격 저장소 연결
* **git push -u origin main**: 온라인 업로드
* **git pull origin main --rebase**: 서버 내용 가져와서 내 작업과 합치기

## 4. 실수 복구 & 트러블슈팅
* **git restore [파일명]**: 수정 내용 취소 (되돌리기)
* **git restore --staged [파일명]**: add 취소
* **git reset HEAD~1**: 방금 한 commit 취소
* **Vim 탈출**: \`Esc\` -> \`:wq\` -> \`Enter\` (저장 후 종료)
* **충돌(Conflict)**: 파일 직접 수정 후 \`add\` -> \`rebase --continue\`
